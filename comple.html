GMTK GAME JAM 2025

Schrodinger's Cat

กระผมและเพื่อนอีกหนึ่งคนได้เข้าร่วม GMTK Game Jam 2025 ใน Theme "Loop" และได้สร้างเกมนี้ขึ้น ภายในระยะเวลา 72 ชั่วโมง โดยออกแบบให้เมื่อแมวตัวนี้ตาย มันจะทิ้งกล่องไว้แล้วกลับมาเกิดที่ spawn point ซึ่งผู้เล่นจะต้องใช้กล่องนี้ในการแก้ปริศนาตามด่านไปเรื่อยๆ แต่ถ้าหากผู้เล่นทำผิดพลาดจะต้องกลับไปเริ่มต้นใหม่ทั้งหมด ทำให้แมวตัวนี้ได้ "วนลูป" ก้าวผ่านความตายนับครั้งไม่ถ้วนเพื่อที่จะกลับไปหาเจ้าของของมัน

Game Jam นี้ทำให้ผมที่รับหน้าที่ การเขียน Code ,ทำ Asset ,และ animation ได้พัฒนาทักษะการทำเกมในเวลาจำกัด และการทำงานร่วมกับผู้อื่นอย่างมาก

CAMT YOUNG GAME DEV

Collage of Arts, Media and Technology

เข้าร่วมการแข่งขันสร้างเกม Young Game Dev ที่ CAMT Open House 2025
กระผมและเพื่อนอีกหนึ่งคนได้เข้าร่วมการแข่งขันนี้ โดยเป็นการแข่งขันสร้างเกมภายในเวลาเพียง 6 ชั่วโมง
แม้ไม่สามารถทำเกมออกมาให้เสร็จสมบูรณ์ได้ แต่ก็เป็นประสบการณ์ที่ดีในการได้ลองทำงานภายใต้ข้อจำกัดด้านเวลา
และทำให้รู้ถึงข้อเสียของตัวเองที่หวังพึ่งปัญญาประดิษฐ์มากเกินไป จึงต้องพัฒนาทักษะการเขียนโปรแกรมให้ดีขึ้น

การแข่งขันสร้างเกมสร้างสรรค์

งานศิลปะหัตกรรม — 2024

กระผมและเพื่อนอีกหนึ่งคนได้เข้าร่วมการแข่งสร้างเกม 6 ชม. โดยใช้ Construct3 ในการสร้างเกม กระผมรับผิดชอบการเขียน block code และ animation
โดยได้รับรางวัลรองชนะเลิศอันดับที่สอง ซึ่งเป็นประสบการณ์ที่ทำให้ผมได้พัฒนาการทำงานเป็นทีม

การแข่งขัน สร้างหุ่นยนต์ผสม

งานศิลปะหัตกรรม — 2023

ผมได้เข้าร่วมการแข่งขันสร้างหุ่นยนต์ กับทีมอีก 3 คน ผมได้รับหน้าที่การออกแบบตัวหุ่นที่สร้างโดยใช้ไม้ และบังคับโดย microcontroller และได้รับรางวัลรองชนะเลิศอันดับที่สอง ทำให้ผมได้พัฒนาทักษะการออกแบบเชิงเทคนิค ที่ใช้ทั้งความคิดสร้างสรรค์และหลักการ
